# Oficina de análise de dados em R do PET-SI


![Logo PET-SI](./images/logo-pet.png "Logo PET-SI")




## Boas-vindas

Seja bem-vindo a oficina de análise de dados do Programa de Ensino Tutorial do curso de Sistemas de Informação (PET-SI).





## Objetivo

Nessa oficina, você irá aprender os fundamentos da análise de dados usando a linguagem R, muito utilizada pelos estatísticos, analistas e cientistas de dados. 





## Público-alvo

- Iniciantes em análise de dados;
- Estudantes da UFSM interessados em aprender o conteúdo;
- Pessoas sem experiência prévia em R;




## Pré-requisitos

- Noções básicas de programação (desejável);
- Alternativamente, utilize o ambiente pronto via GitHub Codespaces (recomendado);
- Ter o R e Rstudio ou correlatos instalado em sua máquina.




## Como usar


### Opção 1: GitHub Codespaces

1. Clique em "Code"
2. Selecione "Codespaces"
3. Crie um novo ambiente
4. Aguarde a inicialização
5. Execute os scripts da pasta `/src/aula-1` e `/src/aula-2`


### Opção 2: Local

```r
# Instalar dependências
install.packages(c('renv', 'tidyverse', 'ggplot2'))
```





## Estrutura da oficina 

Esse repositório contém todo o código extra que pode ser usado pelos alunos antes e depois da oficina. Ele contém todo o código abordado em aula, além de outros exemplos em código, desafios e slide. Esse material pode ser acessado após a oficina para ser usado em sua máquina em casa.

```
./
├── data        -> pasta com os .csv's didáticos
├── README.md   -> este arquivo
├── slides      -> pasta com os slides da apresentação
└── src         -> pasta com os códigos das duas aulas
    ├── aula-1  -> pasta com os códigos da primeira aula
    ├── aula-2  -> pasta com os códigos da segunda aula
```





## Conteúdo

- Introdução ao R;
- Importação de dados;
- Manipulação com dplyr;
- Visualização com ggplot2;
- Análise exploratória.
